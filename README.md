# assisterr-ai-daily-bot

# Features
- Auto login (with mnemonic and/or pkey)
- Auto claim daily

# Requirement
- VS Code
- Node.js 18+ / Bun

# How to run
- Open VS Code and fill mnemonic.txt or pkey.txt with your all SOL mnemonic or private key (separate each line)
- Open terminal
- Command:
  > npm install

  > node index
- Done
